Azure Free Tier Project Report
Introduction
This project involved setting up a cloud environment using Microsoft Azure. The goal was to understand cloud fundamentals such as resource management, identity access, and cost monitoring.

Region Selection
The selected region for this project is West Europe.

This region was chosen because:

It supports Free Tier resources

It provides high availability

It offers better service compatibility for beginners

Although West Africa (Nigeria) is closer geographically, it has limited services compared to West Europe.

Resource Deployed
A Storage Account was deployed as the test resource.

This service allows users to store data in the cloud and represents a basic example of cloud storage infrastructure.

Shared Responsibility Model
In cloud computing, security responsibilities are shared between the cloud provider and the user.

Microsoft Azure is responsible for:

Physical data centers

Hardware and infrastructure

Network security

The user is responsible for:

Data security

Access control (RBAC)

Configuration of resources

This means that while Azure secures the cloud, the user must secure what they put in the cloud.
